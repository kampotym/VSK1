# VSK1

AI-Based Personal Finance Assistant

This project is a simple AI-based personal finance assistant designed to help users track and understand their spending habits. The system automatically categorizes financial transactions and generates a monthly spending summary, reducing the need for manual expense tracking.

Features

Transaction input via backend API (/transactions)

Rule-based transaction categorization (Food, Transport, Entertainment, Other)

Monthly spending summary by category

Simple HTML frontend interface

JSON-based output for financial summaries

Purpose

Many individuals struggle to monitor daily expenses and manage their budgets effectively. This application demonstrates how basic AI logic and automation can simplify personal finance management and provide clearer insights into spending behavior.
